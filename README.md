# Gas Rental — Frontend Starter

## Setup
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## Keys
- LocalStorage keys: `GR_cars`, `GR_users`, `GR_tx`, `GR_auth`
- To reset data, clear these keys in browser DevTools.

## Test checklist
1. Rent Now → Booking wizard → upload KTP preview.
2. Set province ≠ `Jawa Tengah` → Not Available prevent booking.
3. Payment simulates 1.2s → Payment success page.
4. Admin Manage Cars CRUD persists to localStorage.
5. History lists transactions.
